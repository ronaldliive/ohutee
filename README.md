# Ohutee

Ohutee muudab Eesti inimkannatanutega liiklusõnnetuste avaandmed lihtsaks piirkonnapõhiseks vaateks.

Andmeallikas: Transpordiamet / Politsei- ja Piirivalveamet. GitHub Actions kontrollib ametlikku andmestikku iga päev, teisendab selle veebirakendusele sobivaks ning salvestab värskenduse automaatselt.

Ohuskoor kirjeldab ajalooliste õnnetuste koondumist valitud raadiuses. See ei ennusta konkreetset õnnetust ega asenda ametlikku liiklusohutuse hinnangut.
